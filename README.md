# MarketInsight Dashboard

A high-performance market intelligence dashboard built with Vite, React, and Tailwind CSS.

## Features

- **Multi-Asset Tracking**: Monitor Stocks, Crypto, Commodities, and Forex in one place.
- **Real-time Analytics**: Interactive charts powered by Recharts.
- **Modern Dark UI**: Clean, accessible design with glassmorphism effects.
- **Dynamic Filtering**: Quickly search and filter through hundreds of assets.
- **Pro Insights**: Integrated news feed and portfolio summary.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
